#  Ma_reu : une application de gestion de réunions
 
#  développée pour l’entreprise Lamzone en croissance, afin d'améliorer l’organisation des réunions (connaître les salles disponibles et la liste des participants…)

#  Ma_reu est une application responsive développée en Android  et en JAVA 

#  Les fonctionnalités développées : 

#  Afficher les réunions	(avec un visuel et des infos déterminés)
#  Ajouter une réunion (date, salle, participants) 
#  Filtrer les réunions (selon salle ou date)

# application testée par 14 tests d’intégration et tests unitaires
